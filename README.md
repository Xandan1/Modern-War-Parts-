EN:

This mod introduces blocks and parts that complement vanilla elements. The main goal of the mod: It was created for my SM project to add parts missing in the vanilla game, as well as to give these parts 9 durability levels to make armor implementation possible.

Who Can Use This Mod?
 This mod is publicly available, and you are free to use it for regular purposes. HOWEVER, PLEASE NOTE: we update the mod based on our own need for parts. In the event of global updates that drastically change or remove parts—which might prevent your creation from spawning—we highly recommend joining our Discord server so we can assist you.
